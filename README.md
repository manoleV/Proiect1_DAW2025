# Proiect1_DAW2025
primul poriect la DAW, manole vladislav
Este un startup/companie care ajuta studentii/elevii sa ramana concentrati si sa-si faca temele la timp. Scopul principal este de a reduce stresul si de a demonstra ca si invatatul poate fi fascinant si placut.

Situl web este compus din 4 pagini:
	1. Acasa - landingpageul unde este prezentat foarte pe scurt misiunea companiei si un call to action
	2. Servicii - locul unde sunt prezentate 3 pachete basic sau actiuni suplimentare pe care le poate cere studentul.
	3. Despre noi - o scurta descriere fictiva a companiei cu niste review-uri adevarate :)
	4. Sfaturi - aici utilizatorul primeste un test despre productivitatea sa si un sfat personalizat in functie de procentajul luat.

Idea a pornit din viata reala unde nu reuseam nici cum sa tin pas. Nu voiam deloc sa las totul pe maine dar nici azi nu faceam nimic. pare un proiect interesant si vreau sa-l prelungesc. Sper ca pana la finalul modului sa ma pot axa un pic mai tare pe creativitate si sa fac ceva mai misto cu proiectul.
